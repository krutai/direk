Kurulum

  1. supabase.com üzerinde ücretsiz proje aç.
  2. SQL editörüne git, sağdaki SQL'i yapıştır ve çalıştır.
  3. Project Settings → API bölümünden URL ve anon public anahtarını kopyala.
  4. Bu dosyanın en altındaki CONFIG bloğuna yapıştır.
  5. Dosyayı Netlify, Vercel, GitHub Pages gibi bir yere yükle.

Tablo şeması

create table direkmatik_hits (
  id bigserial primary key,
  user_key text not null,
  hit_at timestamptz not null default now()
);
create index on direkmatik_hits (user_key, hit_at desc);

alter table direkmatik_hits
  enable row level security;

create policy "anyone read own"
  on direkmatik_hits for select
  using (true);
create policy "anyone insert"
  on direkmatik_hits for insert
  with check (true);
create policy "anyone delete own"
  on direkmatik_hits for delete
  using (true);
